
## Getting Started

To run application follow these steps.

1. Clone the repo
   ```sh
   git clone https://github.com/maciejewskakatarzyna/transactions_app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the project
   ```sh
   npm start
   ```

To test application follow these steps.
1. To start tests
   ```sh
   npm run test
   ```

2. To start e2e tests
   ```sh
   npm run test:e2e
   ```
